<div align="center">

</div>

---

## 👋 소개

4차산업연구회는 **인공지능, 빅데이터, IoT, 클라우드 컴퓨팅** 등 4차 산업혁명의 핵심 기술을 연구하고 실험하는 모임입니다. 

최신 기술 트렌드를 학습하고, 실제 프로젝트를 통해 경험을 쌓으며, 함께 성장하는 것을 목표로 합니다.

---

## 🎯 주요 활동

- 📚 **기술 스터디**: 최신 AI/ML, 웹개발, 클라우드 기술 학습
- 💻 **프로젝트 개발**: 실무 중심의 다양한 프로젝트 진행
- 🤝 **지식 공유**: 기술 블로그, 세미나, 워크숍 개최
- 🏆 **해커톤 참여**: 대회 참가 및 포트폴리오 구축

---

## 🛠 기술 스택

**주요 관심 기술:**

```
Programming: Python, JavaScript, Java, Go
AI/ML: TensorFlow, PyTorch, Scikit-learn
Web: React, Node.js, Django, FastAPI
Cloud: AWS, Google Cloud, Azure
Database: PostgreSQL, MongoDB, Redis
DevOps: Docker, Kubernetes, CI/CD
```
