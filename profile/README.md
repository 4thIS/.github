# 4차산업연구회

## 개요

## 소속 정보

- 조직명: 4차산업연구회
- 학교: 명지전문대학
- 학과: 정보통신공학과
- 지도교수: 정필성

## 목적

이 조직 GitHub는 동아리 구성원들의 학습 활동, 프로젝트 개발, 세미나 자료, 그리고 기술 연구 내역을 기록하고 공유하기 위해 운영됩니다.
